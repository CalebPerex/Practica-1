Web 1.0

Web 1.0: La Red de Lectura (1990 - 2004)

La Web 1.0 se caracterizó por ser una red estática y unidireccional. Los usuarios eran consumidores pasivos de información, ya que el contenido era creado y administrado por desarrolladores o “webmasters”. Las páginas estaban construidas principalmente con HTML básico, sin bases de datos dinámicas ni interacción avanzada.

La comunicación era limitada: el usuario solo podía leer la información. No existían redes sociales, comentarios ni perfiles personalizados. El diseño era simple y muchas páginas parecían documentos digitales.

Década de 1990: El Nacimiento

  • 1990: Tim Berners-Lee desarrolla el primer servidor web y el lenguaje HTML en el CERN, marcando el inicio de la World Wide Web.
  • 1993: Se lanza Mosaic, el primer navegador gráfico que permitió visualizar imágenes junto al texto.
  • 1995: Surgen empresas como Amazon y eBay, impulsando el comercio electrónico. También aparece JavaScript, permitiendo pequeñas interacciones.
  • 1998: Google introduce PageRank, mejorando la forma de organizar la información en internet.

Década de 2000 (Inicios): Madurez del HTML

  • 2000: Estalla la burbuja punto com, provocando la caída de muchas empresas digitales.
  • 2001: Nace Wikipedia, demostrando el potencial del conocimiento colaborativo, aunque aún bajo estructura técnica sencilla.
